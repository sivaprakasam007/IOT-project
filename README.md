# 🌡 Temperature Controlled Fan using Arduino

## Introduction
Temperature controlling is required in many places such as server rooms, houses, industries, etc. This project demonstrates the basics of temperature control that can be used at home or anywhere. Here, the temperature-controlled fan reacts automatically to temperature changes.

We have also written a blog on temperature sensors. If you are interested, feel free to read it to understand more about these sensors.

We are going to implement this using a **DHT11 temperature and humidity sensor**. It can also be done using thermistors or contactless sensors. However:

- Thermistors usually require surface contact
- Contactless sensors can be expensive
- **DHT11 is cost-effective and beginner-friendly**

## Project Overview
In this project, we build a **temperature-controlled fan using Arduino**.

### Components Used
- Arduino UNO
- DHT11 Sensor Module
- 16x2 LCD Display
- DC Fan

### Working Principle
The system reads room temperature using the sensor and automatically adjusts fan speed accordingly. The current temperature and fan speed are displayed on the LCD screen.
